# blob.ojisan.io

sadnessOjisan のブログ。 技術以外も何でもアリ。

## 開発者向け

皆さんからの PR を受け付けております。

### 技術構成

- React
- Gatsby

なぜ Gatsby を使っているかは、[なぜ私は Gatsby でブログを作っているのか](https://blog.ojisan.io/why-i-will-use-gatsby/) をご覧ください。

### セットアップ

Requirement: Node.js(v16~)

```
npm install

npm run develop
```
